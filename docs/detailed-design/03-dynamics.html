5. 序列圖 (Sequence Diagrams)

5.1 SD-1：使用者上傳 Register Excel

sequenceDiagram
    actor U as User
    participant FE as Frontend
(Registers.tsx)
    participant API as Backend
(registers.py)
    participant EP as ExcelParser
    participant FS as Filesystem
    participant DB as SQLite

    U->>FE: 拖曳 spe_registers_sample.xlsx
    FE->>FE: 驗證副檔名 .xlsx
    FE->>API: POST /api/registers (multipart)
    API->>EP: parse(file_bytes)
    EP->>EP: 讀 sheet "Registers" 第 7 列起
    EP->>EP: 萃取 ADDR/Register/INI/Bits/Member
    EP-->>API: { registers: [...], stats: {...} }
    alt 解析成功
        API->>FS: 儲存 data/registers/{id}/spe.xlsx
        API->>DB: INSERT INTO register_definitions
        API-->>FE: 200 { id, register_count, bitfield_count }
        FE->>U: 列表新增一筆
    else 解析失敗
        API-->>FE: 400 { detail: "..." }
        FE->>U: 顯示錯誤提示
    end

5.2 SD-2：上傳 N 個 dat 並執行分析

sequenceDiagram
    actor U as User
    participant FE as Frontend
(Analyze.tsx)
    participant API as Backend
(batches.py)
    participant FS as Filesystem
    participant ANA as Analyzer
    participant REP as Reporter
    participant DB as SQLite

    U->>FE: 選 Register 下拉
    U->>FE: 拖曳 / 選資料夾 (2000 dat)
    FE->>FE: filter .dat 結尾
    U->>FE: 點「開始分析」
    FE->>API: POST /api/batches
(register_id + 2000 files)

    API->>FS: 寫 data/batches/{id}/dats/*.dat
    API->>API: ExcelParser.load(register_id)
    loop 每個 dat
        API->>API: DatParser.parse(file)
    end
    API->>ANA: analyze(registers, all_dats)
    ANA->>ANA: 為每行 dat 解析 bit field
    ANA-->>API: result_matrix + warnings
    API->>REP: writeCsv + writeXlsx
    REP->>FS: data/batches/{id}/result.csv
+ result.xlsx
    API->>DB: INSERT INTO batches
    API-->>FE: 200 { batch_id }

    FE->>FE: navigate(/results/{id})
    FE->>API: GET /api/batches/{id}
    API->>DB: SELECT batch
    API->>FS: 讀 result.csv → JSON
    API-->>FE: 200 { summary, table, warnings }
    FE->>U: 渲染五個 Tab

graph LR

6.3 DF-3：Registers + Dat → Result Matrix（核心）

graph TB

6.6 DF-6：數值解讀格式 (ValueFormat) 資料流（v0.44）

後端永遠儲存 unsigned 原始整數；FieldRange.format（uint / sint / fp32，存於 rangeMap → localStorage，每個 register 一份）只改變前端怎麼解讀這個原始值。共用 helper interpretValue(raw, width, format) 與 formatBounds(width, format) 定義於 useBitFieldTypes.ts。注意：主資料表儲存格與 2D 熱力圖／散佈圖不經過解讀，維持顯示原始 unsigned 值。
graph TB
    RAW[Raw unsigned int
row.values i
後端原始值] --> FMT{FieldRange.format
uint / sint / fp32}

    FMT --> IV["interpretValue(raw,width,format)
sint = 二補數
raw≥2^(w-1) ? raw-2^w : raw
fp32 = Uint32→Float32 重解讀"]

    IV --> SP[StatsPanel
computeStats
範圍過濾 + 直方圖
bounds = formatBounds 或自訂 min/max]
    IV --> OP[OverallPanel
min/max/不同值/涵蓋率
理論範圍 = formatBounds
fp32 顯示 FP32、涵蓋率 —]
    IV --> OOR[超範圍警告
Results.tsx outOfRangeWarnings
+ ResultsTable 紅色高亮
比較前先解讀]

    RAW -.不解讀.-> TBL[主資料表儲存格
ResultsTable 顯示
原始 unsigned]
    RAW -.不解讀.-> HM[2D 熱力圖 / 散佈圖
Heatmap2D / Scatter
原始 unsigned]

7. 狀態圖 (State Diagrams)

7.1 ST-1：Analyze 頁三步驟狀態機

stateDiagram-v2
    [*] --> Step1_NoRegister
    Step1_NoRegister: Step 1
未選 Register
    Step2_AwaitDat: Step 2
已選 Register
等待 dat
    Step3_Ready: Step 3
可開始分析
    Analyzing: 分析中
(mock 1.2s)
    Done: 完成

    Step1_NoRegister --> Step2_AwaitDat : 選 Register
    Step2_AwaitDat --> Step1_NoRegister : 換 Register / 清空
    Step2_AwaitDat --> Step3_Ready : 加入至少 1 個 dat
    Step3_Ready --> Step2_AwaitDat : 移除全部 dat
    Step3_Ready --> Analyzing : 點「開始分析」
    Analyzing --> Done : 完成
    Done --> [*] : navigate /results/{id}

7.2 ST-2：分析批次 Lifecycle（後端）

stateDiagram-v2
    [*] --> Received
    Received: 收到 multipart 請求
    Validating: 驗證 register_id
+ 檔案格式
    Parsing: ExcelParser
+ DatParser
    Analyzing: Analyzer 比對
    Reporting: 寫 CSV/XLSX
    Persisting: INSERT batches
    Done: 回應 200
    Failed: 回應 4xx/5xx

    Received --> Validating
    Validating --> Failed : 缺欄/格式錯
    Validating --> Parsing
    Parsing --> Failed : Excel 解析失敗
    Parsing --> Analyzing
    Analyzing --> Reporting
    Reporting --> Persisting
    Persisting --> Done
    Done --> [*]
    Failed --> [*]

7.3 ST-3：Bit Field 類型

stateDiagram-v2
    [*] --> CheckStorage
    CheckStorage: useBitFieldTypes init

    HasStorage: 有存檔
    NoStorage: 無存檔

名稱含 EN/MODE

否則 magnitude

    CheckStorage --> HasStorage : key 存在
    CheckStorage --> NoStorage : key 不存在

        ResetDefaults --> ModeMagOthers
    }
